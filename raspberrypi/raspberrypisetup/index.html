Raspberry Pi Setup

The Raspberry Pi is a tiny $35 computer. It is sold without any components, just a simple circuitboard. One would expect that such a cheap computer would be second rate. On the contrary, it provides more than enough power. Because of its tiny size, it can be used for many different projects, such as a personal server or a robotics project. Here we'll outline setting one up and a few things to do with it.

Collecting the materials

For starters, check out the Raspberry Pi web site They have some great documentation to get you familiar with the concept. The model of Raspberry is up to you, but I would highly suggest the latest Raspberry Pi 3 because it has much more capabilites than the previous models and it has built-in wifi. That in itself is a huge plus. Once you've ordered a Pi from one of their vendors, you need to consider its accessories. It will need some sort of a power cord. The Pi's official charger is Micro USB. (For information about the differences between micro and mini usb, check here) You will definitely want a high-quality cord that is capable of outputting at least 5 volts and 2 amps. I would suggest the offical Raspberry pi cord. The other thing you will want to consider is a miniSD card. Since the Raspberry Pi won't boot without one, this is something you will want to seriously consider. When I went to buy an card, I went for the biggest one I could find, a $5 64GB miniSD card from a random seller on eBay. What I quickly found was that there are differences between cards. Eventually, I had to buy an 8GB preloaded SD card from adafruit.com. One thing to beware of when buying preloaded card, however, is that only Raspbian Jessie (the latest version) will work in the Raspberry Pi 3. Raspbian Wheezy will not.

There are a few optional items that you may want. They are an HDMI cable and a TV or HDMI monitor. If you plan on running your Pi as a server, these are unnecessary. However, if you want a media center, you will definitely want to take time planning out these things, as well as a USB mouse and keyboard.. If you do decide to go without the display, you can still access your Pi's screen through SSH and VNC, which we will explain later. One other thing you will need is an internet connection either directly to your router with an ethernet cable, or via wifi. If you choose to use wifi, you will need a monitor and HDMI cable to set it up. If you have a wired connection to your router, the monitor is unnecessary. For the purpose of this tutorial, I'll assume you are choosing to connect headlessly, or without a monitor.

Plugging things in

Once you have all the materials, it is time to start hooking things up. Start by placing the miniSD card into the small reader on the under-side of the Pi, then plug in the ethernet cable. Finally, plug the power cable into the port on the Pi and plug the other end into the outlet. At about this time you should see a couple lights on the Pi, namely a red and a green one. The green one should be blinking; it indicates SD card activity. The red one should not; it indicates that there is power to the Pi.

SSH

The first thing you will have to do is to connect to the Pi via SSH. To do you need to access your router's setup page. To accomplish that, I would highly recommend this guide from How-To Geek. If you absolutely cannot access your router's setup page, try the AngryIP scanner.

Windows

If you have Windows, you will need to download PuTTY, a free download that will enable you to access your Raspberry Pi's terminal from your Windows PC. When it finishes downloading, open the exeecutable. Enter your Pi's IP address into the "Host name" bar and click "Open."

Macintosh

Open the Terminal from the Apple menu. Then, at the prompt, type, 

ssh pi@your_Pi's_IP_address

Linux

The instructions for Linux are very similar to the Macintosh instructions. Simply open the Terminal (usually located under "System" or "Accessories") and type 

ssh pi@your_Pi's_IP_address

Android

In the Google Play Store, search for "JuiceSSH." Install and open it. Once it is open, tap "Connections." Hit the yellow + sign in the lower right-hand corner. Type a name for the connection in the top field (I chose "raspi"). Then type your Pi's IP address in the "Address" field. In the "Identity" field hit "New," then type another name in the top field. Type "pi" in the "Username" field. Then tap the checkmark in the top right corner. Finally, tap the checkmark icon in the upper-right corner to finish making the connection. At this stage, you can hit "All" and then hit "raspi" and it will connect.

Updating the Raspberry Pi

If it asks you for login credentials; the username is "pi" and the password is "raspberry." Once you're logged in, you will first want to type

sudo raspi-config

Select the first option, "Expand the Filesystem." When that has finished, you will want to select the second option, "Change User Password." It will ask you to type your new password twice. Once it is complete, select "Finish." Let it restart your Pi. When it restarts, you will lose your SSH connection to it. Wait about 30 seconds and then try to reconnect to it. When you get connected, log on, then type

sudo apt-get update

and 

sudo apt-get upgrade

Once that is complete, you have a fully working Raspberry Pi. You can now proceed to start a project.
